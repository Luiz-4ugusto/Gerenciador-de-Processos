# Gerenciador-de-Processos

# 📝 Sistema de Lembretes em Python

Projeto simples de **gerenciamento de lembretes com notificações no desktop**, desenvolvido em Python utilizando interface gráfica.

O objetivo do projeto é permitir que o usuário configure lembretes personalizados que aparecem como **notificações no sistema operacional** em intervalos definidos.

## 🚀 Funcionalidades atuais

* Criar lembretes personalizados
* Definir título e mensagem da notificação
* Escolher quantidade de repetições
* Definir intervalo entre notificações
* Interface gráfica simples usando Tkinter
* Notificações no desktop utilizando Plyer
* Execução de lembretes em paralelo com Threads

## 🛠 Tecnologias utilizadas

* Python
* Tkinter
* Plyer
* Threading

## 📂 Estrutura básica do projeto

```
📁 projeto-lembretes
 ├── main.py
 ├── centraliza.py
 └── Forest-ttk-theme
```

## ▶️ Como executar

1. Instale as dependências:

```
pip install plyer
```

2. Execute o programa:

```
python main.py
```
